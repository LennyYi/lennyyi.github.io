这是根据github以及jekyll来完成的个人静态网站，也是我自己第一个网站。
jekyll提供了很强大的语法让我们来随心所欲的构建体现自己风格的网站下面是段代码

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
